\section{Introducción}

GitHub Issues es una herramienta que permite gestionar incidencias, errores, mejoras y tareas dentro de un proyecto de software. Su uso facilita la organización del trabajo colaborativo, ya que permite asignar responsabilidades, dar seguimiento a las actividades y documentar la solución de problemas.

En esta actividad se realizó la administración de incidencias mediante la interfaz web de GitHub y mediante GitHub CLI desde la terminal Git Bash. Para ello se simuló un entorno de trabajo colaborativo en el repositorio, donde se crearon incidencias, se visualizaron, se asignaron responsables y se documentó su seguimiento.
